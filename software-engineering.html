School Programming Projects

Word Frequency Finder

This word frequency finder has one C++ file that uses object oriented programming to find the most used words in a text. This program uses sorting algorithms, mainly merge sort. There are two outputs this program produces. The first output is a .txt file that has all the words in the book sorted alphabetically with their frequency next to them. This output is called the “bookname”.txt – alphabetical.txt. The second output file is all the words sorted by their word frequency. The top word is the word that is used the most in the entire book. This file is outputted as “bookname”.txt – frequency.txt. This project is an assignment from the University of Arkansas and contains modified code from John Gauch.

Name Finder

This program written in C++ uses a binary search tree in order a name that the user asks from a database containing the 1,000 most popular names for boys and girls in the United States. If the name is not found, the program will tell the user that the name was not found and ask for another name to search. If the name is found, then it will return the name, a percentage of popularity, and the rank of the name. This program has code that belongs to John Gauch. This was a project for my Programming Foundations 2 class at the University of Arkansas.

Front-End Developer

I am currently developing personal website using HTML, CSS and JavaScript, learning fundamental principles of web development and site deployment. I Deployed website using AWS EC2 Ubuntu with NGINX as principal server and HTTPS handler. o Setup HTTPS for the site using Certbot and Let’s Encrypt.

This has been a great personal project because I am learning about website development and getting better on working towards my personal goals. Developing a website has also taught me how to think. If you know HTML, you know DIVs work. If you don’t, let me explain briefly. A parent DIV is like a container. For this example, we have a DIV that is a lunch bag . Our goal is to fill the lunch bag with food! We cannot just throw pasta, an apple and juice in there, Each different item needs its own container. The pasta has its own smaller container, a child DIV! The apple can be wrapped in a napkin, and the juice has its own box. Each container is different for each item in the lunch bag. Each child DIV is different than the previous one. So, we started with a problem, filling the lunch bag, and we broke it down in smaller problems. Now you can enjoy your lunch! I translate this way of thinking to real-life problems, homework assignments, and even cooking. Ilook at the big picture (parent div), then break down each step there is to take (child divs). This is an extremely helpful skill to have because now I can solve problems faster, easier, and more reliably.
